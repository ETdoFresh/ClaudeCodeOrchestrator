# Claude Code Orchestrator

A .NET 8 Avalonia desktop application for orchestrating Claude Code sessions with git worktree support.

## Under Construction

This project is actively under development. Features and APIs may change.

**Pull requests to improve functionality are greatly appreciated!**

## What is this?

Claude Code Orchestrator provides a way to manage and orchestrate Claude Code workflows:

- **Parallel Tasks** - Send multiple tasks to Claude Code simultaneously
- **Long-Running Jobs** - Set up and monitor extended Claude Code sessions

## Getting Started

```bash
cd src/ClaudeCodeOrchestrator.App
dotnet run
```

## Tech Stack

- .NET 8
- Avalonia UI (cross-platform desktop)
- LibGit2Sharp for git operations
